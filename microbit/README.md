## version 1.x

## btlejack
https://github.com/virtualabs/btlejack

1. `pip install btlejack`
1. connect all microbits (3x)
1. `btlejack -i` -> install latest firmware
1. start capturing: `btlejack -d /dev/ttyACM0 -d /dev/ttyACM1 -d /dev/ttyACM2 -c any`
